// El proceso

PIPELINE DE LA REFINERÍA

4 pasos. Completamente autónomo. El cliente sube basura digital, recibe un Video Coherente de Pensamiento.

01
📥
CAPTURA
Webhook de n8n recibe los clips (mp4, webm, mkv), el guion y los metadatos del cliente vía formulario, Telegram o Google Sheets.
n8n Webhook
02
🧠
ANÁLISIS
Claude actúa como tasador de chatarra. Extrae la esencia narrativa, asigna clips por emoción (Tensión, Calma, Revelación) y genera el JSON estructurado.
Claude API · claude-sonnet-4
03
🎬
REFINERÍA
Remotion renderiza el JSON como componente React: overlays de texto, transiciones, pacing de audio. Lambda serverless — $0.00 si no hay pedidos.
Remotion Lambda · AWS
04
📤
ENTREGA
n8n envía el video al cliente con la factura automática. Mensaje: "Tu chatarra ha sido refinada. Tarifa de Reciclaje Digital aplicada: $0.35."
n8n · Stripe · Email

// Stack tecnológico

  
LA MAQUINARIA

  
Open source. Sin dependencias de pricing externo. Soberanía de costos total.

  

    

      🧠
      
Claude

      
El Arquitecto. Analiza guiones, asigna clips por emoción, genera JSON estructurado para Remotion.

      
claude-sonnet-4 · $0.003/análisis

    

    

      🎬
      
Remotion

      
Motor de video en React. Renderiza el JSON de Claude como componentes visuales con transiciones y overlays.

      
Lambda serverless · $0.002/video

    

    

      🔄
      
n8n

      
El capataz. Orquesta el pipeline completo: entrada → Claude → Remotion → factura → entrega.

      
Self-hosted · $0/mes

    

    

      ☁️
      
AWS Lambda

      
Renderizado serverless. $0.00 cuando no hay pedidos. Escala automáticamente con la demanda.

      
Pay per use · ~$0.002/render

    

    

      🎞️
      
Pexels API

      
Stock de clips gratuito para el Plan Estándar. 50K+ videos de alta calidad listos para la refinería.

      
Free tier · sin atribución requerida

    

    

      ⚡
      
Vite + React

      
Frontend del panel de control. Componentes Remotion se construyen aquí antes del deploy.

      
Open source · $0
